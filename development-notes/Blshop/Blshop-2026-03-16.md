## 2026-03-16

### Code Review

Разбирался с **React Router**, с помощью которого была создана маршрутизация в проекте by **GorodeN**.
Так как с React я сталкиваюсь впервые, для меня всё было абсолютно новым. Сразу разобраться не получилось.

Я попытался изучить официальную документацию, но это не дало ожидаемых результатов: код проекта сильно отличался от примеров из документации.
Пришлось обратиться к искусственному интеллекту для построчного разбора кода. Это уже дало результаты: после пары дней изучения я более-менее разобрался, как это работает.

Дельного комментария к ПР сделать не удалось из-за недостатка знаний, но навыки, полученные при изучении кода, были использованы при разработке страницы с вопросами — и всё прошло без сучка и задоринки.

Далее изучал создание компонентов от **DmitryAstapenko**.
Компоненты были реализованы на React, что опять оказалось для меня вызовом. Сразу использовал ИИ для построчного разбора кода. Код напоминал то, что проходили на курсе, и полный разбор вариантов создания элементов занял меньше времени, чем ожидалось.

Идеи, полученные при изучении этих компонентов, были использованы при создании собственного элемента **HighlightedText** на основе существующего **Text**, для подсветки кодовых слов в тестах.
