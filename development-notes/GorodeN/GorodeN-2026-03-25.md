Приступил к написанию кода `server` (подсматривал в видео на Youtube - [Автор: Ulbi TV | Продвинутая JWT авторизация на React и Node js. Access, refresh, активация по почте](https://www.youtube.com/watch?v=fN25fMQZ2v0)).

1. Добавил и установил зависимости `server`:

- `bcryptjs` - для хеширования паролей
- `cookie-parser` - для извлечения данных из куки и их удобного использования в виде объекта
- `cors` - механизм защиты
- `dotenv` - для загрузки переменных окружения из `.env` для изоляции конфигурации
- `express` - фреймворк для веб-сервера
- `jsonwebtoken` - для JWT токенов
- `mongoose` - либа для работы с MongoDB
- `nodemon` - для отслеживания изменений в коде и автоматической перезагрузки сервера

2. Реализовал файлы:

- `server/.env` - переменные окружения
- `server/.env.example` - примеры переменных окружения
- `server/src/config/env.js` - загрузка переменных окружения
- `server/src/config/db.js` - подключение к MongoDB
