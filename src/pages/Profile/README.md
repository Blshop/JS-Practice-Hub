# Profile Page

Страница профиля пользователя с визуализацией статистики прохождения уроков.

## Компоненты

### UserCard

Карточка пользователя с основной информацией:

- Аватар (первая буква username)
- Username и email
- Заработанный XP / Общий XP
- Пройдено уроков / Всего уроков

### StatsCards

4 карточки с ключевыми метриками:

- Total Attempts (всего попыток)
- Successful (успешных попыток)
- Failed (неудачных попыток)
- Success Rate (процент успеха)

### AttemptsChart

Круговая диаграмма (Pie Chart) распределения успешных/неудачных попыток.
Использует Recharts.

### ModulesChart

Столбчатая диаграмма (Bar Chart) прогресса по модулям.
Показывает количество пройденных уроков в каждом модуле.
Использует Recharts.

### LessonsTable

Таблица со статистикой по каждому уроку:

- Название урока и модуля
- Успешные/неудачные попытки
- XP награда
- Прогресс-бар с процентом успеха

## Хук useProfileStats

Обрабатывает данные из `mock-user-server-progress.json` и `js-learning-path-data.json`:

- Подсчитывает общую статистику
- Группирует данные по модулям
- Сортирует уроки по количеству попыток
- Вычисляет заработанный XP

## Источники данных

- `src/data/mock-user-server-progress.json` - статистика с сервера
- `src/data/js-learning-path-data.json` - информация об уроках
- `authStore.user` - данные текущего пользователя
